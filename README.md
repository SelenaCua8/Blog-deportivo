# 🏆 Blog Deportivo : RedditDeportes

Este es el primer avance del proyecto final para la materia **Producción Web** de la carrera **Analista de Sistemas** (Escuela Da Vinci). El proyecto consiste en la planificación, diseño de interfaces y modelado de base de datos para un blog temático de deportes.

## 📝 Información del Proyecto
* **Nombre del Proyecto:** RedditDeportes
* **Temática:** Blog de noticias, análisis y resultados deportivos.
* **Objetivo:** Brindar un espacio interactivo donde los fanáticos del deporte puedan informarse, debatir a través de comentarios y seguir a sus redactores favoritos.
* **Público Dirigido:** Aficionados al deporte en general, desde fútbol hasta disciplinas olímpicas.

## 🚀 Funcionalidades Principales
- **Sistema de Roles:** Gestión de permisos diferenciados para Visitantes, Usuarios Registrados, Editores y Administradores.
- **Gestión de Contenidos:** Publicación de artículos con categorías y etiquetas (hashtags).
- **Interacción:** Sistema de comentarios y gestión de perfiles de usuario.
- **Panel de Administración:** Dashboard completo para la gestión de usuarios y reportes del sitio.

## 📂 Estructura del Repositorio
El repositorio está organizado siguiendo los requerimientos del TP1:

- `/HTML`: Contiene todas las maquetas funcionales del sitio.
  - `/index`: Vistas públicas (Login, Register, Post, etc).
  - `/user`: Vistas exclusivas para usuarios registrados.
  - `/editor`: Gestión de contenidos y categorías.
  - `/administrador`: Gestión de usuarios y configuración global.
- `/Diagramas`: Documentación técnica de la base de datos.
  - `DER_Blog_Deportivo.pdf`: Diagrama Entidad-Relación.
  - `Modelo_Logico_Blog.pdf`: Estructura lógica de tablas y relaciones.
- `Consigna_TP1.pdf`: Documento con los requerimientos de la facultad.

## 🛠️ Tecnologías Utilizadas
- **Frontend:** HTML5, CSS3, JavaScript.
- **Modelado de Datos:** Draw.io / MySQL Workbench.
- **Control de Versiones:** Git & GitHub.
